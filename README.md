## Problem 5:Write a C program that takes a student's score (0-100) as input and prints the corresponding grade based on the following rules:

### Grading Criteria:
- 90-100 -> Grade A
- 80-89  -> Grade B
- 70-79  -> Grade C
- 60-69  -> Grade D
- 0-59   -> Grade F

### Additional Requirements:
- If the user enters a number less than 0 or greater than 100, print `"Invalid score"`
- Use only `if`, `else if`, and `else` statements (no loops or switch statements)

### Example Output:
- Input: `85` -> Output: `Grade B`
- Input: `102` -> Output: `Invalid score`

### Hint:
You will need multiple `else if` conditions to check each score range carefully.
